# PrankGuard 

Webcam-based security system with face recognition, USB device detection, and automatic screen locking.

## Features
- Face recognition (owner vs threat)
- Instant USB/HID device detection
- Monitor, Network, Bluetooth, Printer, Audio detection
- Automatic USB port blocking
- Shoulder surfer detection
- Camera disconnect protection

## Requirements
- Python 3.12+
- Windows 10/11 (Admin for USB blocking)
- Webcam

## Install
`pip install opencv-python face-recognition customtkinter keyboard pywin32 wmi Pillow`

## Run (admin required)
`python prankguard.py`
